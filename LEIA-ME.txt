============================================
 WhatsBot - Bot WhatsApp com IA
============================================

REQUISITOS:
  - Windows 10/11
  - Python 3.11 ou superior (https://python.org)
  - gowa.exe (baixe de https://github.com/nicnocquee/gowa/releases)

SETUP RAPIDO:
  1. Instale o Python 3.11+ (marque "Add to PATH" no instalador)
  2. Baixe o gowa.exe e coloque na pasta bin\
  3. Execute run_dev.bat para rodar em modo desenvolvimento
     OU
     Execute build.bat para gerar o executavel

COMO USAR:
  1. Ao abrir, o WhatsBot inicia o servidor e abre o navegador
  2. Escaneie o QR Code com seu WhatsApp (Dispositivos Conectados)
  3. Configure sua API Key do OpenRouter (https://openrouter.ai)
  4. Defina o System Prompt (personalidade do bot)
  5. Clique "Salvar Configuracoes"
  6. O bot comeca a responder automaticamente!

ESTRUTURA:
  whatsbot/
  ├── main.py          - Ponto de entrada
  ├── server/          - Backend web (FastAPI)
  ├── web/             - Frontend web (Preact + Tailwind)
  ├── gowa/            - Gerenciamento do GOWA (WhatsApp)
  ├── agent/           - Agente IA (OpenRouter)
  ├── config/          - Configuracoes
  ├── bin/             - Binario GOWA (gowa.exe)
  ├── build.bat        - Script para gerar EXE
  ├── run_dev.bat      - Rodar em modo dev
  └── requirements.txt - Dependencias Python

CONFIGURACOES:
  Salvas em: %APPDATA%\WhatsBot\config.json
  Logs em:   %APPDATA%\WhatsBot\logs\

PROBLEMAS COMUNS:
  - "gowa.exe nao encontrado": Baixe e coloque em bin\gowa.exe
  - "API key invalida": Verifique sua chave em openrouter.ai
  - QR Code nao aparece: Aguarde o GOWA iniciar (pode levar 10s)
  - SmartScreen bloqueia: Clique "Mais informacoes" > "Executar assim mesmo"
